# DS1 Editor Examples

This directory contains practical examples demonstrating various DS1 map editing techniques.

## Available Examples

### Act 1 Examples

**Act1-Town-4Objects-Example** ([Files](Act1-Town-4Objects-Example/) | Data: `Act1-Town-4Objects-Example-Data/`)
- Demonstrates how to add 4 objects to the Act 1 Town area. Referenced in [Adding Monsters and Objects Guide](../03-Advanced-Guides/Adding-Monsters-And-Objects-Guide.html)
- Shows object placement and integration techniques
- Good for learning basic object editing

### Act 3 Examples  

**Act3-Town-Method1-Example** (Archive: `Act3-Town-Method1-Example.zip`, Data: `Act3-Town-Method1-Example-Data/`)
- Shows Act 3 town editing using the first method
- Demonstrates advanced town modification techniques

**Act3-Town-Method2-Example** (Archive: `Act3-Town-Method2-Example.zip`, Data: `Act3-Town-Method2-Example-Data/`)
- Alternative approach to Act 3 town editing
- Compares different methodologies for the same result

### Boss Area Examples

**Doe-Frozenstein-Example** (Archive: `Doe-Frozenstein-Example.zip`, Data: `Doe-Frozenstein-Example-Data/`)
- Example of editing the Frozenstein boss area
- Demonstrates boss room modification techniques
- Version 1.13 compatible

### Construction Examples

**House-Building-Example** (Archive: `House-Building-Example.zip`, Data: `House-Building-Example-Data/`)
- Step-by-step house construction tutorial
- Shows architectural editing techniques
- Includes structural planning and execution

**Tent-Construction-Example** (Archive: `Tent-Construction-Example.zip`, Data: `Tent-Construction-Example-Data/`)
- Demonstrates tent object construction
- Covers temporary structure placement
- Useful for camp and settlement editing

### Integration Examples

**Text-And-DS1-Integration** ([Files](Text-And-DS1-Integration-Data/) | Archive: `Text-And-DS1-Integration.zip`)

- Shows how to integrate text files with DS1 maps. Referenced in [Tutorial 01](../02-Tutorials/Tutorial-01-Basic-Map-Editing.html)
- Demonstrates metadata and configuration usage
- Advanced integration techniques

## How to Use Examples

1. **Extract Archives**: Each `.zip` file contains the complete example data
2. **Browse Data Folders**: The `-Data` folders contain extracted/working versions
3. **Study Techniques**: Each example demonstrates specific DS1 editing approaches
4. **Apply Learning**: Use these examples as reference for your own projects

## Example Structure

Each example typically contains:
- DS1 map files
- Associated data files (DT1, etc.)
- Configuration or text files
- Sometimes screenshots or documentation

## Integration with DS1 Editor

- Load example DS1 files directly in the editor
- Use as reference when testing new features
- Compare your results with provided examples
- Launch configurations can reference these examples
